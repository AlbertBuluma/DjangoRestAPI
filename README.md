# DjangoRestAPI
A sample Django API demo.

### Get started

``` 
$ git clone https://github.com/AlbertBuluma/DjangoRestAPI.git
$ pip install -r requirements.txt
```

### Run tests:

``` 
$ python manage.py test
```
